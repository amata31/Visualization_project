# Data Visualization Projects

These projects are part of the IBM *Data Visualization with Python* course. They focus on creating visualizations, interactive maps, and dashboards using Python.

## Projects

### 1. Data Visualization with Python
Exploratory data visualization using Python libraries.

**Libraries used:**
- Matplotlib
- Seaborn
- Folium
- Pandas
- NumPy

**Topics covered:**
- Line plots
- Bar charts
- Histograms
- Scatter plots
- Heatmaps
- Interactive maps

### 2. Automobile Sales Dashboard with Dash
Interactive dashboard built with Dash and Plotly to analyze automobile sales during recession and non-recession periods. This is an app ready to be deployed.

**Libraries used:**
- Dash
- Plotly
- Pandas

**Features:**
- Dropdown filters
- Yearly statistics
- Recession period statistics
- Interactive line charts, bar charts, and pie charts
