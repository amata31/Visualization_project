# Data Visualization Project

This project is part of the IBM *Data Visualization with Python* course. It focuses on creating visualizations and interactive maps using Python libraries to explore and analyze data.

## Libraries Used
- Matplotlib
- Seaborn
- Folium
- Pandas
- NumPy

## Topics Covered
- Line plots
- Bar charts
- Histograms
- Scatter plots
- Heatmaps
- Interactive maps
